?? PERSONAL ACCESS TOKEN

¿Por qué lo necesito?
  ? GitHub requiere autenticación moderna
  ? No puedes usar tu contraseña en la terminal

¿Cómo crearlo?

  1. Ve a: https://github.com/settings/tokens
  2. Click: "Generate new token" ? "Generate new token (classic)"
  3. Configura:
     - Name: NexShop-Upload
     - Expiration: 90 days
     - Scopes: ? repo (marcar esto)
  4. Click: "Generate token"
  5. COPIA EL TOKEN (aparece solo una vez)

?? SOLUCIONAR PROBLEMAS

Problema: "fatal: remote origin already exists"
Solución:
  git remote remove origin
  git remote add origin https://github.com/TU_USUARIO/NexShop.git
  git push -u origin main

Problema: "Authentication failed"
Solución:
  - Verifica que tu token no está expirado
  - Crea un token nuevo en: https://github.com/settings/tokens
  - Asegúrate de copiar el token correctamente

Problema: "Permission denied (publickey)"
Solución:
  - Usa HTTPS en lugar de SSH
  - git remote set-url origin https://github.com/TU_USUARIO/NexShop.git

Problema: "git: no se encontró el comando"
Solución:
  - Descarga Git desde: https://git-scm.com/download
  - Reinicia la terminal después de instalar
