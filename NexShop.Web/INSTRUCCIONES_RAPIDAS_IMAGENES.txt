?????????????????????????????????????????????????????????????????????????????

? SIN DISTORSIÓN
   • aspect-ratio: 1/1 (siempre cuadrado)
   • object-fit: contain (sin corte)
   • Centrado automático

? RESPONSIVE AUTOMÁTICO
   • No necesita ajustes manuales
   • Se adapta automáticamente a cualquier pantalla
   • Desde 360px hasta 4K+

   • Lazy loading (imágenes cargan bajo demanda)

   • Modo oscuro soportado
   • Reduce motion respetado
   • Navegación por teclado

? ROBUSTO
   • Fallback si no hay imagen
   • Manejo automático de errores
   • Sincronización inteligente


?? TROUBLESHOOTING
?????????????????????????????????????????????????????????????????????????????

? Las imágenes no aparecen
   ? Verifica que estén en: wwwroot/uploads/productos/{id}_{nombre}/
   ? Ejecuta el script de sincronización nuevamente
   ? Recarga la página (Ctrl+F5)
   ? Revisa la consola del navegador (F12)

? Script de sincronización falla
   ? Abre PowerShell como Administrador
   ? Verifica que la ruta sea correcta
   ? Ejecuta: powershell -ExecutionPolicy Bypass
   ? Intenta nuevamente

? La aplicación no inicia
   ? Verifica: dotnet --version (debe ser 8.0 o superior)
   ? Instala dependencias: dotnet restore
   ? Limpia: dotnet clean
   ? Compila: dotnet build

? Las imágenes se ven distorsionadas
   ? Borra caché del navegador (Ctrl+Shift+Delete)
   ? Abre en incógnito
   ? Verifica que las imágenes sean válidas


?? API ENDPOINTS
?????????????????????????????????????????????????????????????????????????????

Obtener imágenes de un producto:
  GET /api/imagenes/producto/1
  
  Retorna:
  [
    {
      "nombre": "imagen1.jpg",
      "url": "/imagenes/productos/1_Smartphone/imagen1.jpg"
    }
  ]

Sincronizar archivos:
  POST /api/imagenes/sincronizar
  
  Retorna:
  {
    "mensaje": "Sincronización completada exitosamente",
    "carpetasProcesadas": 50,
    "imagenesCopiadas": 150,
    "errores": 0
  }

Sincronizar multimedia en BD:
  POST /api/imagenes/sincronizar-multimedia
  
  Retorna:
  {
    "mensaje": "Multimedia sincronizada exitosamente",
    "productosActualizados": 50,
    "multimediaAgregada": 150,
    "errores": 0
  }
