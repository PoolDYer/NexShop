?? RESUMEN

Se han creado 3 scripts para crear carpetas de productos automáticamente.

?? CÓMO USARLO

PASO 1: Abre PowerShell
  ? Clic derecho en la carpeta NexShop.Web
  ? "Open in Terminal" o "Open PowerShell here"
  OR
  ? Cmd/PowerShell manual: cd "E:\Proyectos Visual\NexShop\NexShop.Web"

PASO 2: Ejecuta el script
  ? .\crear-todas-carpetas.ps1

PASO 3: Listo!
  ? Se crearán carpetas para TODOS tus productos

?? SIGUIENTES PASOS

1. Ejecutar script crear-todas-carpetas.ps1

2. Abrir Visual Studio:
   Solution Explorer
   ??? wwwroot > uploads > productos
   
3. Descargar imágenes de internet:
   • Google Imágenes
   • unsplash.com
   • pexels.com
   • pixabay.com
   
4. Colocar imágenes en cada carpeta:
   1_Producto/ ? imagen1.jpg, imagen2.jpg, video.mp4
   2_Producto/ ? imagen1.jpg, imagen2.jpg
   etc...

5. ¡Listo! Las imágenes aparecerán en la app automáticamente
